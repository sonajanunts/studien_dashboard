## Studien Dashboard
 
Python-Projekt für das Portfolio im Kurs  
„Objektorientierte und funktionale Programmierung mit Python“.

## Funktionen

- Verwaltung von Studienmodulen
- Anzeige von ECTS und Notenschnitt
- Berechnung des Studienfortschritts
- Fortschrittsbalken für Zeit- und Studienfortschritt
- Speicherung der Daten mit SQLite
- Grafische Benutzeroberfläche mit tkinter

## Verwendete Technologien

- Python 3.11
- tkinter
- sqlite3

## Projekt starten

Im Projektordner ein Terminal öffnen und folgenden Befehl ausführen:
```bash python main.py
